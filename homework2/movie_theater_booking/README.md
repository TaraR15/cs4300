To run the Movie Theater Booking System locally navigate to the project directory with cd cs4300/homework2/movie_theater_booking. Create and activate a Python virtual environment using python -m venv venv followed by source venv/bin/activate, then install all dependencies with pip install -r requirements.txt. Set up the database by running python manage.py migrate, and optionally create a superuser for admin access using python manage.py createsuperuser. If you want sample movie and seat data, run the seeding script with python seed.py. Finally, start the development server with python3 manage.py runserver 0.0.0.0:3000 and access the application at with the app button in devedu, the admin panel by adding /admin to the end of the url. To run tests, use python manage.py test bookings -v 2 for unit and API tests, or python manage.py behave for BDD tests. 

The application is also live on Render at https://movie-theater-booking-fd92.onrender.com/
However, this will not have the movie or seat data as I had issues accessing the admin page and cannot use the shell with the free version, but the base template is live.
